# Rex Explorer - Inquisitive Investigators

## Features

- File browser
- Built-in terminal
- Ease-of-use footer buttons
- Text editor

## Installation

#### Users

- Install pipenv: `pip install pipenv`
- Install dependencies: `pipenv sync`

#### Developers

- Install pipenv: `pip install pipenv`
- Install dependencies: `pipenv sync --dev`
