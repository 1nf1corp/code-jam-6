## Setup

##### pipenv
```
python -m pipenv install
```
or to include `flake8`
```
python -m pipenv install --dev
```

##### pip

```
python -m pip install requirements.txt
```

note that `flake8` is not included

## Running the project

- `pipenv run start`
- `python -m project`
